# 电商平台UI设计规范

## 设计理念
参考京东、淘宝的成熟电商设计，结合简约现代风格，打造专业、易用的购物平台。

---

## 4. 首页设计

### 顶部导航栏
```
+---------------------------------------------------------------+
|  [Logo]  [首页] [商品分类] [我的订单] [购物车] [个人中心]    [搜索框......] [登录/注册] |
+---------------------------------------------------------------+
```

- 高度: 60px
- 背景: 白色
- Logo: 左侧，距左边 20px
- 导航菜单: 左侧，文字 #333333
- 搜索框: 宽度 400px，圆角 20px，右侧放大镜图标
- 登录按钮: 右侧，主色背景白色文字

### 轮播Banner
```
+---------------------------------------------------------------+
|  [商品分类侧边栏]  [========== 轮播图 (5张) ==========]       |
|                  [  1   2   3   4   5  ]                      |
+---------------------------------------------------------------+
```

- 左侧分类: 宽度 200px，背景 #F7F7F7
- 轮播图: 宽度 800px，高度 400px
- 指示器: 底部居中，圆点样式

### 快捷入口
```
+---------------------------------------------------------------+
|  [京东秒杀]  [京东直播]  [京东超市]  [京东家电]  [京东生鲜]    |
+---------------------------------------------------------------+
```

- 4个图标 + 文字卡片
- 圆角矩形，每个宽度均匀分布

### 商品推荐模块
```
+---------------------------------------------------------------+
|  为你推荐                                                    |
+---------------------------------------------------------------+
|  +-------+  +-------+  +-------+  +-------+  +-------+       |
|  |       |  |       |  |       |  |       |  |       |       |
```

## 12. 动画效果

### 过渡动画
- 全局: 0.3s ease
- 卡片hover: transform scale(1.02), box-shadow加深

### 加载动画
- 商品骨架屏: 灰色渐变动画
- 按钮点击: 轻微缩放

---

## 总结

本设计规范融合了京东和淘宝的优点:
1. **红色主色调** - 借鉴京东，增强品牌识别
2. **简洁卡片设计** - 借鉴淘宝，突出商品
3. **清晰的层级** - 信息层次分明
4. **流畅的交互** - 细腻的动画反馈

请开发团队按照此规范实现UI。
